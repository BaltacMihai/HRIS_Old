# HRIS 
## How to set-up

### Front-end
```
cd client
npm i
```

### Back-end
```
cd server
npm i
```

### Database
Open a MySql server on local, create a database with the name: **hris**

## How to open

### Front-end
```
cd client
npm start
```

### Back-end
```
cd server
nodemon
```

To create the tabels for the app, please use the route **/reset**
